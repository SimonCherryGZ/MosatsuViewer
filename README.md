
MosatsuViewer
===================================  
  本应用是Mosatsu（妄撮）风格图片的浏览工具。
  网上流传的类似应用多数只能浏览自带的若干组图片，而本应用支持用户导入自己的图片。
   
     
应用平台 
-----------------------------------  
  Android
